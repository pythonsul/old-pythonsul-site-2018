Python Sul 2017


### Set up & Install dependencies

```
$ git clone git@github.com:pythonsul/pythonsul-site.git
$ cd pythonsul-site
$ gem install jekyll
```

### Server

```
$ jekyll serve
```

### Gulp

```
# To install gulp deps
$ npm install

# If you need to rebuild javascript
$ gulp
```

### Using a browser, go to:

```
http://localhost:4000
```
